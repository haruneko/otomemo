## タグ語彙の暫定ドラフト（要確認＝これで意味/雰囲気/様式検索を動かせる）
意味検索の足切りに使う「閉じた語彙」案。研究＝感情は arousal(強度)＞valence(明暗) が安定なので粗い軸で。
**誰がタグ付け**＝(a) 自動導出できるものは自動（終止＝cadenceOf／明暗＝長短比＋テンション量）、(b) mood/genre は取込時に Claude が NL→この語彙へ写像、(c) ユーザー上書き可。
- **明暗(valence)**：`明るい` / `中間` / `切ない`（=暗い・哀）
- **強度(arousal)**：`静か` / `普通` / `激しい`（＋「切羽詰まった」≒切ない×激しい の合成で表現）
- **ジャンル/様式(genre)**：`J-POP` / `ロック` / `バラード` / `シティポップ` / `ダンス/EDM` / `劇伴/アンビエント` / `ジャズ寄り`（粗く・複数可）
- **セクション役割(section)**：`イントロ` / `Aメロ` / `Bメロ` / `サビ` / `ブリッジ/Cメロ` / `アウトロ`
- **終止(cadence・自動)**：authentic/plagal/half/deceptive（cadenceOf が算出・タグ不要）
要確認：軸/項目の粒度、ジャンル項目の取捨、「アーティストっぽさ」は別軸(指紋)にするか語彙に混ぜないか。
