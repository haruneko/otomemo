# UI 語彙 triage（2026-07-22・全 UI 語を一律の物差しで再評価）

## 経緯・基準
- Fable UX監査（2026-07-22）でオーナーが承認した修正を実装する中で、オーナーから語彙への指摘が入った：
  > 「『キックに噛む』って日本語で言う？（英語の lock with the kick の直訳では）」
- **裁定＝Fable監査の好評は判定根拠にしない**。全ての UI 語を**同じ物差し**で問い直す。
- 物差し（厳しめ）：**「日本語として実際に使われる言い回しか」**。
  - 英語圏の音楽イディオムの直訳（lock / pocket / bite / in the pocket 系）＝分類1。
  - 自動詞/他動詞の転用（「噛み合う」→「〜に噛む」型の造語的他動詞化）＝分類1。
  - オノマトペ/内輪の愛称＝分類1（悪いとは限らないが表に載せて判断を仰ぐ）。
- **合否はオーナー triage に委ねる**（開発は審判あり・創作/命名まわりは物差しがオーナー＝正しさを簒奪しない）。
  今回のコード改名は**オーナー既裁定分（②ストローク・③速さ・⑥左手ラベル短縮・⑦候補を出す）に限定**。本表の分類1/2はコードを変えず提案に留める。

## 分類
- **1＝造語・内輪語・直訳**（日本語話者が音楽の場で自然には使わない／要検討）
- **2＝専門ジャーゴンだが定着**（音楽をやる人には通じる借用語・術語）
- **3＝素直な日本語**（問題なし）

## 「キックに噛む」の代替案（オーナー指定で2案・逆相整合込み）
1. **キックに合わせる**
   - 長所：日本語として自然・意図（キックにタイミングを合わせる）が直裁。
   - 逆相整合の注意：「合わせる」の段に「逆相（＝わざと外す）」があるのは語感が逆。→ 採る場合は**逆相段を「わざとずらす／裏に置く」に改称**して整合を取る。
2. **キック連動**
   - 長所：名詞で中立・強度段（弱/強）と相性が良い。**逆相＝負の連動**として矛盾なく収まる（「連動」の反対＝逆相という読みが自然）。
   - 短所：やや硬い（説明的）。副文「キックにベースを乗せる」で補える。
